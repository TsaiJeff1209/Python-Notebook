## 表格
| 欄位名稱 | 描述 |
|---|---|
| name | 專案名稱(與專案目錄同名) |
| packages | 要安裝的套件名稱 |
| scripts | script名稱，通常代表一個執行檔，不一定有 |
| version | 版本 |
| description | 專案描述 |
| author | 作者 |
| author_email | 作者信箱 |
| keywords | 這個專案的一些關鍵字 |
